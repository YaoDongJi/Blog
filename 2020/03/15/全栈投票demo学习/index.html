链接 https://www.jianshu.com/p/b282d375e45e

  1. 配置区块链本地运行环境，之前装过了python安装工具和web3，ganache_cli,安装命令 cnpm install ganache-cli web3@0.20.0

  2. 项目架构

  3. 编译和部署一个合约的步骤

    1. 在node环境下获得web3和solc实例

    2. 读取代码作为变量，solc示例编译产生一个contract对象（包含abi和字节码信息）

    3. 创建合约对象

      1. 通过contract对象获取abi字符串变量

      2. 获得知道abi的合约对象，和字节码信息

      3. 用以上信息部署合约，需要传入构造函数参数

      4. 获取合约实例
        contractInstance = VotingContract.at(deployedContract.address)

  4. 使用实例调用合约接口

    可以理解成navigator.mediaDevices.getUserMedia(constrains) 返回一个参数stream ，然后执行function功能
