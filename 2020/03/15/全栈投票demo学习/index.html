全栈投票demo学习

第一部分

链接 https://www.jianshu.com/p/b282d375e45e

  1. 配置区块链本地运行环境，之前装过了python安装工具和web3，ganache_cli,安装命令 cnpm install ganache-cli web3@0.20.0
  2. 项目架构
  3. 编译和部署一个合约的步骤
    1. 在node环境下获得web3和solc实例
    2. 读取代码作为变量，solc示例编译产生一个contract对象（包含abi和字节码信息）
    3. 创建合约对象
      1. 通过contract对象获取abi字符串变量
      2. 获得知道abi的合约对象，和字节码信息
      3. 用以上信息部署合约，需要传入构造函数参数
      4. 获取合约实例

        contractInstance.totalVotesFor.call(‘Rama’)
        { [String: ‘0’] s: 1, e: 0, c: [ 0 ] }
        contractInstance.voteForCandidate(‘Rama’, {from: web3.eth.accounts[0]})
        ‘0xdedc7ae544c3dde74ab5a0b07422c5a51b5240603d31074f5b75c0ebc786bf53’
        contractInstance.voteForCandidate(‘Rama’, {from: web3.eth.accounts[0]})
        ‘0x02c054d238038d68b65d55770fabfca592a5cf6590229ab91bbe7cd72da46de9’
        contractInstance.voteForCandidate(‘Rama’, {from: web3.eth.accounts[0]})
        ‘0x3da069a09577514f2baaa11bc3015a16edf26aad28dffbcd126bde2e71f2b76f’
        contractInstance.totalVotesFor.call(‘Rama’).toLocaleString()
        ‘3’
        ```

  4. 使用合约index.html时要修改index.js里的合约地址为自己自己合约实例地址
